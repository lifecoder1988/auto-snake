# auto-snake

自动玩贪吃蛇思路

关于身体碰撞的检测

需要考虑前进了N步，小蛇的尾巴就少了N个。也就是说 撞上了如果发现小蛇的身体部位属于后半截N，就表示没撞上

关于安全定义

小蛇当前状态下 走曼哈顿回路不会碰撞到自己的身体

游戏目标

吃完所有苹果

其他注意事项

- 不能穿墙
- 速度最快 10ms移动一步 

当前状态下小蛇有2个选择

1. 去吃苹果，有两种选择

- 走最短路径

  - 规划一条最短路径， 与小蛇身体不碰撞,需要确保吃到苹果后还是安全的。

- 走最长路径

  - 与小蛇身体不碰撞

2. 回到环路上去，有两种选择

- 立即回到大环路

- 往前走一步再判断
